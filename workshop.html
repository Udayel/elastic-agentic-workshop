☁️ Step 1: Get Elastic Cloud via AWS Marketplace

💡 Why AWS Marketplace?
  • Pay via AWS account (unified billing)
  • Use AWS promotional credits
  • 7-day free trial included
  • Simplified procurement
1 Subscribe to Elastic Cloud
# 1. Open AWS Console # 2. Search for "Elastic Cloud" in AWS Marketplace # 3. Click "Continue to Subscribe" # 4. Click "Set up your account" # Direct link: https://aws.amazon.com/marketplace/pp/prodview-iyc3k5vglqxve
2 Create Deployment
  • Name: travel-agent-workshop
  • Version: 8.15.0 or latest
  • Provider: AWS
  • Region: us-east-1 (match your Lambda region!)
  • ML nodes: 4GB RAM (required for ELSER)
  • ES nodes: 8GB RAM x 2 zones
3 Save Credentials ⚠️ IMPORTANT
⚠️ Password shown only once!
Download CSV or save to password manager immediately.
ELASTIC_CLOUD_ID: "your-deployment:dXMtZW..." ELASTIC_ENDPOINT: "https://abc123.es.us-east-1.aws.found.io:9243" ELASTIC_PASSWORD: "AbCdEfGh..." ← SAVE THIS!
4 Deploy ELSER Model
  • Open Kibana → Machine Learning → Trained Models
  • Find ".elser_model_2"
  • Click "Download model" (1-2 min)
  • Click "Deploy model"
  • Wait for status: "Started" ✅
5 Request Bedrock Access
  • AWS Console → Amazon Bedrock
  • Left menu → Model access
  • Click "Request model access"
  • Select: Claude 3.5 Sonnet, Claude 3 Haiku
  • Submit (usually instant approval)

❓ Troubleshooting

Connection timeout to Elastic

Solution: Add your IP to Elastic traffic filter

# Get your IP: curl https://api.ipify.org # Add to: Elastic Cloud Console → Security → Traffic filters
ELSER model not found

Solution: Download and deploy in Kibana

  1. Kibana → ML → Trained Models
  2. Search: ".elser_model_2"
  3. Click "Download model"
  4. Click "Deploy model"
  5. Wait for "Started" status
Bedrock access denied

Solution: Request model access

  1. AWS Console → Bedrock → Model access
  2. Click "Request model access"
  3. Select Claude models
  4. Wait 2-5 minutes
